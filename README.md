## Español

Este repositorio documenta mi proceso de aprendizaje en el lenguaje de programación Java,
combinando los contenidos del curso universitario de Informática con estudio
autodidacta a través de un curso especializado en Udemy.

El objetivo no es solo aprender a programar, sino construir una base sólida y documentada
que refleje progresión real, desde los fundamentos hasta conceptos más avanzados.

**Fuentes:**
- Universidad de Londres — Fundamentos de Programación
- Universidad Java - Cero a Experto — Udemy
